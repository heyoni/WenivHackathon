# 용역 서류 자동 생성 서비스 - 요구사항 명세서

## 1. 프로젝트 개요

### 1.1 목적
용역 계약의 **제안 → 계약 → 착수 → 완료** 단계별로 필요한 서류들을 관리하고 자동 생성하는 웹 서비스.

### 1.2 핵심 컨셉
```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  제안   │ → │  계약   │ → │  착수   │ → │  완료   │
│         │    │         │    │         │    │         │
│ 과업지시서│    │ 안내/체크│    │ 착수계  │    │ 완료계  │
│ (AI생성) │    │ 리스트  │    │ 인력투입 │    │ 완료검사 │
│         │    │         │    │ 공정예정 │    │ 완료내역 │
│         │    │         │    │ 보안각서 │    │         │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
```

### 1.3 핵심 기능
1. **프로젝트 생성**: 새 용역 프로젝트 생성
2. **단계별 진행**: 제안 → 계약 → 착수 → 완료 순서로 진행
3. **서류 자동 생성**: 기본 정보 입력 시 모든 양식에 자동 적용
4. **AI 과업지시서**: 간단한 설명만 입력하면 AI가 과업지시서 초안 생성
5. **PDF 출력**: 단계별 서류 일괄 다운로드

## 2. 기술 스택

```
Frontend: React + Vite
```

## 3. 단계별 서류 구조

### 3.1 제안 단계 ⭐ (AI 활용)

#### 과업지시서
- **사용자 입력**:
  - 과업명
  - 예산
  - 수행기간 (시작일 ~ 종료일)
  - 간단한 설명 (1-2문장)
  - 대상
  - 예상 결과물

- **AI 자동 생성**:
  - 과업목적
  - 과업범위 및 내용
  - 수행방법
  - 일정계획

- **문서 구조**:
  ```
  [표지]
  - 「{과업명}」 용역 과업지시서
  
  [1. 과업개요]
  - 과업명, 예산, 목적, 수행기간, 개요
  
  [2. 세부사항]
  - 과업범위 및 내용 (AI)
  - 수행방법 (AI)
  - 일정계획 (AI)
  
  [3. 과업보고]
  - 착수/결과 보고, 결과물 제출
  
  [4. 일반사항]
  - 고정 텍스트
  ```

### 3.2 계약 단계

#### 안내 페이지 (서류 작성 X, 체크리스트)
- **업체 제공 서류** (다운로드 링크):
  - 계약보증금지급각서
  - 청렴서약서
  - 서약서
  - 수의계약각서

- **고정 서류** (미리 업로드해두면 출력):
  - 인감증명서
  - 사업자등록증
  - 법인등기부등본
  - 통장사본

- **정부수입인지**: 전자납부 안내

- **입력 항목**:
  - 계약금액
  - 계약년월일

### 3.3 착수 단계

#### 서류 목록 (6종)
1. **착수계**
2. **인력투입계획서** - 참여연구원 표
3. **공정예정표** - 월별 간트차트
4. **책임연구원(기술자) 선임계**
5. **보안각서**
6. **원가계산서** (선택)

### 3.4 완료 단계

#### 서류 목록 (3종)
1. **완료계**
2. **완료검사원**
3. **완료내역서** - 항목별 표

### 3.5 대금청구 단계 (안내만)

- 대금청구서
- 납세증명서
- 4대보험 완납증명서
- 세금계산서 발급

→ 경영지원팀 업무로 상세 안내만 제공

### 5.1 프로젝트 (Project)
```javascript
{
  id: string,
  name: string,                    // 프로젝트명 (= 과업명)
  status: 'proposal' | 'contract' | 'start' | 'complete',
  createdAt: string,

    aiGenerated: {

      schedule: string,            // 일정계획
    }
  },
  
  // 계약 단계 데이터
  contract: {
    amount: string,                // 계약금액
    amountKorean: string,          // 계약금액 (한글)

    checklist: {                   // 체크리스트
      계약보증금지급각서: boolean,
      청렴서약서: boolean,
      서약서: boolean,
      수의계약각서: boolean,
      인감증명서: boolean,
      사업자등록증: boolean,
      법인등기부등본: boolean,
      통장사본: boolean,
      정부수입인지: boolean,
    }
  },
  
  // 착수 단계 데이터
  start: {
    startDate: string,             // 착수년월일
    endDate: string,               // 완료예정년월일
    submitDate: string,            // 제출일
    researchers: [...],            // 참여연구원
    leadResearcher: {...},         // 책임연구원
  },
  
  // 완료 단계 데이터
  complete: {
    completeDate: string,          // 완료년월일
    items: [...],                  // 완료내역 항목
  },
  
  // 회사 정보 (공통)
  company: {
    name: string,                  // 상호
    address: string,               // 주소
```

## 6. AI 과업지시서 생성

### 6.1 사용자 입력
```
- 과업명: 성인학습자 취·창업 역량강화 실무 교육
- 예산: 20,000,000원
- 기간: 2025.11.27 ~ 2026.01.08
- 설명: 바이브코딩을 활용한 AI 교육 프로그램
- 대상: 성인학습자 30명
- 결과물: 교육 수료자, 결과보고서
```

### 6.2 AI 프롬프트
```
당신은 용역 과업지시서 전문 작성자입니다.
다음 정보를 바탕으로 과업지시서의 주요 섹션을 작성해주세요.

[과업명]: {task_name}
[예산]: {budget}원
[수행기간]: {period_start} ~ {period_end}
[간단한 설명]: {description}
[대상]: {target}
[예상 결과물]: {expected_output}

JSON 형식으로 작성:
{
  "purpose": "과업목적 (2-3개 항목)",
  "scope": "과업범위 및 내용 (5-7개 항목)",
  "method": "수행방법 (단계별)",
  "schedule": "일정계획 (주차별/단계별)"
}

작성 시 주의사항:
- 공문서 어투 (~하여야 한다, ~으로 한다)
- 구체적이고 실현 가능한 내용
- 예산 규모에 맞는 현실적인 계획
```
