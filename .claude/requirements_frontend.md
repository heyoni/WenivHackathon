# 용역 서류 자동 생성 서비스 - 요구사항 명세서

## 1. 프로젝트 개요

### 1.1 목적
용역 계약 시 필요한 서류들을 자동으로 생성하는 웹 서비스. 기본 정보를 한 번 입력하면 모든 양식에 자동 적용되어 PDF로 출력.

### 1.2 핵심 기능
- 기본 정보 입력 (계약건명, 금액, 날짜, 회사정보 등)
- 8종 서류 자동 생성 및 미리보기
- PDF 일괄 다운로드
- 데이터 로컬 저장 (localStorage)

---

## 2. 기술 스택

```
Frontend: React + Vite
Styling: Tailwind CSS
PDF 생성: react-pdf 또는 html2pdf.js
Storage: localStorage
배포: Vercel 또는 Netlify
```

---

## 3. 생성할 문서 목록 (8종)

### 착수 단계 서류
1. **착수계**
2. **인력투입계획서** - 참여연구원 표 포함
3. **공정예정표** - 월별 간트차트 형식
4. **책임연구원(기술자) 선임계**
5. **보안각서**

### 완료 단계 서류
6. **완료계**
7. **완료검사원**
8. **완료내역서** - 항목별 표 포함

---

## 4. 데이터 구조

### 4.1 기본 정보 (formData)
```javascript
{
  // 계약 정보
  계약건명: string,
  계약금액_숫자: string,      // "20,000,000"
  계약금액_한글: string,      // "이천만"
  계약년월일: string,         // "2025. 11. 27."
  착수년월일: string,
  완료년월일: string,
  
  // 회사 정보 (기본값 설정 가능)
  상호: string,              // "주식회사 위니브"
  주소: string,              // "제주도 제주시 구산로 58, 2층"
  대표자: string,            // "이호준"
  
  // 제출일
  제출년: string,
  제출월: string,
  제출일: string,
}
```

### 4.2 참여연구원 (researchers)
```javascript
[
  {
    기관구분: string,
    성명: string,
    직급: string,
    생년월일: string,
    학교: string,
    취득년도: string,
    전공: string,
    학위: string,
    담당분야: string,
  },
  // ... 추가 가능
]
```

1. **계약 정보**: 계약건명, 금액(숫자/한글), 계약/착수/완료 년월일

## 7. PDF 생성

### 7.1 방법
- html2pdf.js 또는 react-pdf 사용
- 각 문서별 개별 PDF 생성
- 전체 문서 일괄 다운로드 (ZIP 또는 병합 PDF)

### 7.2 PDF 설정
```javascript
{
  format: 'a4',
  margin: { top: 25, right: 25, bottom: 25, left: 25 },
  filename: '{문서명}_{계약건명}.pdf'
}
```

## 8. 데이터 저장

### 8.1 localStorage 활용
```javascript
// 저장
localStorage.setItem('formData', JSON.stringify(formData));
localStorage.setItem('researchers', JSON.stringify(researchers));

// 불러오기
const savedFormData = JSON.parse(localStorage.getItem('formData'));
const savedResearchers = JSON.parse(localStorage.getItem('researchers'));
```

### 8.2 자동 저장
- 입력값 변경 시 자동 저장 (debounce 적용)
