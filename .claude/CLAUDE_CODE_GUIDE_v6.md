# 견적서 기능 추가 가이드

## 목표
제안 단계(step 1)에 "견적서" 문서를 추가하여, 항목별 비용을 입력하고 A4 형태로 프리뷰/출력할 수 있게 함

## 견적서 구조 (엑셀 참고)

### 카테고리별 항목
1. **인건비**
   - 운영 총괄: N명 × M개월 × 단가
   - 멘토: N명 × M시간 × 단가
   - 강사: N명 × M시간 × 단가

2. **임차비** (단일 항목)

3. **자문비**
   - 자문비 항목: N건 × M주 × 단가

4. **인쇄제작비**
   - 현수막: N종 × M개 × 단가
   - 포스터: N종 × M장 × 단가
   - X-배너: N종 × M개 × 단가
   - 디자인비: N건 × 단가

5. **식다과비**
   - 다과비: N일 × M명 × 단가

6. **행사부대비용**
   - 온라인 콘텐츠 제공: N식 × M명 × 단가
   - 사무용품: N식 × 단가
   - 기타 운영비: N식 × 단가

### 계산
- 소계총합 = 각 카테고리 소계 합
- 부가세 = 소계총합 × 10%
- 합계 = (소계총합 + 부가세) 만원 단위 절사

## 구현 방식

### 1. 데이터 구조 (formData에 추가)
```javascript
// 견적 데이터 (JSON 문자열로 저장)
견적서_항목: JSON.stringify([
  { category: '인건비', name: '운영 총괄', qty1: 1, unit1: '명', qty2: 1, unit2: '개월', price: 2500000 },
  { category: '인건비', name: '멘토', qty1: 3, unit1: '명', qty2: 12, unit2: '시간', price: 100000 },
  // ...
]),
견적서_견적일: '2025-01-01',
견적서_유효기간: '견적일로부터 3개월',
```

### 2. stepDocuments 수정
```javascript
1: [
  { id: '제안서', name: '제안서' },
  { id: '견적서', name: '견적서' },  // 추가
],
```

### 3. 견적서Preview 컴포넌트
- A4Page 컴포넌트 사용
- 테이블 형태로 항목 표시
- 소계, 부가세, 합계 자동 계산

### 4. 입력 패널
- 제안 단계에서 selectedDoc === '견적서' 일 때 입력 폼 표시
- 카테고리별 접이식 섹션
- 항목 추가/삭제 버튼
- 실시간 합계 계산

## UI 설계

### 입력 패널 (왼쪽)
```
[견적 정보]
- 견적일: [날짜 선택]
- 유효기간: [입력]

[인건비] ▼
  운영총괄: [N]명 × [M]개월 × [단가]원 = 자동계산
  + 항목 추가

[임차비] ▼
  임차비: [금액]원

[자문비] ▼
  ...

[소계: xxx원]
[부가세(10%): xxx원]
[합계: xxx원]
```

### 프리뷰 (오른쪽)
- 위니브 견적서 양식 그대로
- 상단: 견적서 제목, 공급자 정보
- 중단: 항목 테이블
- 하단: 소계, 부가세, 합계

## 확인 필요 사항
1. 항목 추가/삭제 기능 필요 여부
2. 공급자 정보(회사명, 주소 등) 수정 가능 여부
3. 비고란 추가 여부
